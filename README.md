# unse.news

정확한 사주(24절기·KST) 계산과 오늘의 운세/궁합/로또 추천을 제공하는 React + Vite SPA입니다.

## 개발

```bash
cd app
npm ci
npm run dev
```

## 테스트/빌드

```bash
cd app
npm test
npm run build
```

## 배포 (GitHub Pages)

- GitHub Actions 워크플로우: `.github/workflows/pages.yml`
- 빌드 산출물: `app/dist`
- 커스텀 도메인: `app/public/CNAME` → `unse.news`

## 라이선스

본 저장소의 코드 및 설정은 저장소 내 명시된 라이선스를 따릅니다.
